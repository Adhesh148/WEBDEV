Live Time Table

About the Project

Live Time Table allows students to view their everyday schedule including:

  1. Regular Classes (excluding cancelled ones)
  2. Extra Classes planned for the week
  3. Scheduled Events such as:
    • Talks
    • Workshops
    • Cultural or Technical Fests
A Live Time Table would help the student better plan his time and also keep track of events ocurring in the college.

Features

We have planned to implement the following features:

Login

  • Student User:

    1. View of the Live Time Table which includes:
      • Regular Classes
      • Additional Classes as updated by the Faculty
      • Also excludes cancelled classes
    2. Events scheduled would be integrated with the daily time table
    3. Student weekly statistics

  • Faculty User:

    1. View of the Student Live Time Table
    2. View of the Faculty Live Time Table
    3. Schedule Class/Event at an avialable time and location
    4. Cancel Scheduled Class/Event (if faculty has required privileges for the particular class/event)

Use Case

Consider the situation that a professor for the COE/CED-18 decides to cancel the 9:00 A.M class on a particular day. This creates a free slot at 9:00 AM. Considering the traditional timetable system no other professor would know of the free slot unless informed by the CR. The traditional system demands the need for human-human communication which could be unreliable at certain times. Our system eliminates the need for human-human communication. An update/deletion on the time time table would be immediately reflected online. This enables the professors and students to plain their day accordingly. For instace, the professor taking the 8:00 A.M. class could shift his class to this interval without the need for any intimidation by the CR. It also saves on sending hundreds of mails by allowing students to view the updated time table live.
